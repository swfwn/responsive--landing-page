# Adventure Travel Landing Page

A responsive landing page built to practice CSS positioning, overlays, and smooth UI effects.

## Concepts Covered

- HTML page structure
- CSS Flexbox and Grid
- Responsive design with media queries
- Positioning (relative and absolute)
- Layering using z-index
- Image overlay using pseudo-elements (::before)
- Hover effects
- Smooth animations using transition and transform
- Opacity-based fade effects

## Features

- Hero section with background image and dark overlay
- Floating overlay card
- Responsive destination grid
- Interactive cards and buttons

This project focuses on layout control and visual interaction using pure CSS.
